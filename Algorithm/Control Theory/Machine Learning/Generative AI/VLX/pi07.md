# 改进

- 多模态 prompt 消除多源数据冲突
	- task instruction
	- subtask
	- subgoal images（基于 BAGEL 的 WM 生成）
	- metadata
	- control mode
- 采用 metadata 记录低质量数据+正确标注，并蒸馏加入了 RL specialist 采用 RL policy rollout 做数据蒸馏，提高了鲁棒性
- 采用 subtask、subgoal images、task instruction 组合技能
- 采用 subgoal 做局部规划（state → future state → action），缩短了state直接到action的跨度

# 成果

1. 无需微调（finetune）有专家（specialist）效果
2. 跨机器人迁移（zero-shot）
3. 通过组合技能完成任务
4. 语言教学
